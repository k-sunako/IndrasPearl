#+title: IndrasPearl

書籍「インドラの真珠: クラインの夢みた世界」に登場する図形や写像を、Python で描画するためのプロジェクトです。

* プロジェクト概要

このリポジトリでは、主に次のような対象を扱います。

- メビウス変換
- 円の像の計算
- ショットキー群に関連する円の配置
- ショットキー群の「ダンス」的な円の反復描画
- 円反転による図形の可視化

実装の中心は =src/lib.py= にあり、以下のような機能を提供しています。

- =MobiusTransform= による 1 次分数変換の表現
- 点に対するメビウス変換
- 円に対するメビウス変換
- 固定点の計算
- 円反転の描画補助
- ショットキー円の像をたどる =schottky_dance=
- その結果を描画する =plot_schottky_dance=

また、実行用スクリプトとして =schottky_dance.py= を用意しており、ショットキー円の図を表示できます。

* 実行に使う主なファイル

- =src/lib.py=
  - 数学的な処理と描画用ヘルパーをまとめたライブラリ
- =main.py=
  - 円反転の図を表示する簡易エントリーポイント
- =schottky_dance.py=
  - =schottky_dance= を使ってショットキー円の図を描画する実行スクリプト
- =pyproject.toml=
  - 依存関係とプロジェクト設定

* 使い方

** 円反転の図を描く

#+begin_src bash
uv run main.py
#+end_src

** ショットキー円の図を描く

#+begin_src bash
uv run schottky_dance.py --depth 3
#+end_src

=--depth= を大きくすると、より深い世代まで円をたどります。
